# Book Tracker

A full-stack book tracking web application built with Node.js, Express, EJS, and MySQL.

## Tech Stack
- Node.js + Express
- EJS (server-side rendering)
- MySQL (PlanetScale)
- JWT Authentication
- bcrypt password hashing
- Deployed on Vercel

## Features
- User authentication (JWT)
- Secure password storage
- Search books via Open Library API
- Add/remove books from personal collection
- View other users' public profiles

## Deployment
- Backend + frontend deployed on Vercel
- Database hosted on PlanetScale
- Environment variables configured in Vercel dashboard
